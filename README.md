# SAMII-TCR

Attention-based meta-learning framework for predicting T-cell receptor (TCR) responses to peptide-MHC (pMHC) complexes.

## Environment Setup

### Requirements

- Python >= 3.9
- CUDA (recommended for GPU acceleration)

### Installation

```bash
# 1. Create conda environment
conda create -n samiitcr python=3.10
conda activate samiitcr

# 2. Install PyTorch (adjust CUDA version as needed)
# For CUDA 12.x:
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu124
# For CPU only:
# pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu

# 3. Install dependencies
pip install -r requirements.txt
```

### Verify Installation

```bash
python -c "import torch; print(f'PyTorch {torch.__version__}, CUDA: {torch.cuda.is_available()}')"
```

## Input Format

The input CSV requires the following columns:

| Column       | Description                                     | Required |
|--------------|-------------------------------------------------|----------|
| `cdr3_TRA`   | TCR alpha CDR3 amino acid sequence              | At least one of TRA/TRB |
| `cdr3_TRB`   | TCR beta CDR3 amino acid sequence               | At least one of TRA/TRB |
| `epitope`    | Peptide epitope sequence                        | Yes      |
| `MHCA`       | MHC-II alpha chain allele (e.g., `DRA*01:01`)   | Yes      |
| `MHCB`       | MHC-II beta chain allele (e.g., `DRB1*04:01`)   | Yes      |
| `Target`     | Binary label: 1 = positive, 0 = negative        | No (for evaluation only) |

See `example/input.csv` for a complete example.

## Quick Start: Prediction

### Hybrid Prediction (Recommended)

Automatically routes seen tasks to Baseline and unseen tasks to Meta model:

```bash
cd models

python predict.py \
  --meta_train ../data/splits_meta_with_neg/exp2_healthy_100x/meta_train.csv \
  --test_file ../example/input.csv \
  --baseline_dir ../trained_models/exp2_baseline_100x \
  --meta_dir ../trained_models/meta_100x_5fold \
  --output ../example/output.csv \
  --device cuda
```

Arguments:
- `--meta_train`: Path to `meta_train.csv`, used to identify which tasks were seen during training
- `--test_file`: Input CSV to predict on
- `--support_file`: *(Optional)* Support set CSV for test-time fine-tuning (TTFT) on few-shot tasks
- `--baseline_dir`: Directory containing 5-fold baseline model checkpoints
- `--meta_dir`: Directory containing 5-fold meta model checkpoints
- `--output`: Path to save prediction results
- `--device`: `cuda` or `cpu`

### Baseline-Only Prediction

```bash
cd models

python predict_baseline.py \
  --input_csv ../example/input.csv \
  --output_csv ../example/output_baseline.csv \
  --variant exp2_healthy_100x \
  --model_root ../trained_models
```

Arguments:
- `--variant`: Training variant (`exp2_healthy_10x`, `exp2_healthy_100x`, `exp2_healthy_1000x`)
- `--model_root`: Parent directory of model folders (expects `train_{variant}/fold_{1-5}/best_model.pt`)

## Output Format

The output CSV contains all input columns plus:

| Column        | Description                                    |
|---------------|------------------------------------------------|
| `hybrid_prob` | Predicted binding probability (sigmoid output) |

Higher `hybrid_prob` indicates higher predicted likelihood of TCR-pMHC interaction.

## Model Architecture

**AttBaselineModel** uses three separate Transformer encoders with attention pooling:

- **TCR Encoder**: Self-attention over CDR3 amino acid sequence
- **Peptide Encoder**: Self-attention over epitope peptide
- **MHC Encoder**: Self-attention over MHC pseudo-sequence

The three pooled vectors are concatenated and passed through an MLP classifier to produce a binary binding prediction.

The meta-learning variant (trained via `train_meta.py`) fine-tunes the last Transformer layer and classifier head using a Reptile outer loop with L2 regularization to prevent catastrophic forgetting, enabling few-shot adaptation to unseen pMHC tasks.

## Supported MHC-II Alleles

DRA, DRB1, DRB3, DRB4, DRB5, DQA1, DQA2, DQB1, DQB2, DPA1, DPB1
